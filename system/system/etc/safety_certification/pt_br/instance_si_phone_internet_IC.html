Conformidade com o Ministério de Inovação, Ciência e Desenvolvimento Econômico do Canadá

Informações de certificação (SAR)

Este dispositivo também foi projetado para atender aos requisitos de exposição a ondas de rádio estabelecidos pelo Ministério de Inovação, Ciência e Desenvolvimento Econômico do Canadá (ISED).

O limite de SAR adotado pelo Canadá é de 1,6 W/kg em média sobre um grama de tecido. O maior valor de SAR informado ao ISED para este tipo de dispositivo está em conformidade com este limite.

Declaração do ISED

Este dispositivo está em conformidade com CAN ICES-003 (B)/NMB-003(B).

Este dispositivo contém transmissor(es)/receptor(es) isento(s) de licença que cumprem com RSS(s) isento(s) de licença do Ministério de Inovação, Ciência e Desenvolvimento Econômico do Canadá. A operação está sujeita às duas condições a seguir:

1. Este dispositivo não pode causar interferência.

2. Este dispositivo deve aceitar qualquer interferência, incluindo interferência capaz de operação indesejada do dispositivo.
